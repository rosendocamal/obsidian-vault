# Fedora Linux

Distribución de [[linux]] patrocinada por Red Hat (ahora IBM). Conocida por su enfoque en la innovación y el software reciente. Es una de las distribuciones más populares para escritorio y desarrollo.

## Características

- Ciclo de lanzamiento rápido (~6 meses), con las últimas versiones de software
- Gran integración con GNOME
- Repositorios vastos y comunidad activa
- Base de Red Hat Enterprise Linux

## Configuración común en Fedora

- Gestor de paquetes: `dnf` (reemplazo moderno de `yum`)
- Gestión de energía con TLP para laptops
- Instalación de VirtualBox con firma de módulos para Secure Boot
- Gestión de batería con `tlp` y trucos para carga parcial

## Relaciones

- Distribución de [[linux]], dentro del ecosistema de código abierto
- Utilizado por el autor como sistema operativo principal
- Aparace en guías: [[guia-fedora-primeros-pasos]], [[guia-virtualbox-fedora]]
- Vinculado a la comunidad de [[red-hat]] y al proyecto [[gnu]]
