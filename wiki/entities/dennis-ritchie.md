# Dennis Ritchie

Científico computacional estadounidense (1941–2011). Creó el lenguaje de programación C (1972) y co-creó el sistema operativo UNIX en Bell Labs.

## Aportaciones clave

- **Lenguaje C (1972)**: lenguaje de propósito general, base de UNIX y de la mayoría de los sistemas modernos
- **UNIX**: sistema operativo que influyó en Linux, macOS y otros
- Su lenguaje C es la base de C++, Python, Java y muchos lenguajes modernos

## Relaciones

- Su lenguaje C fue extendido por [[Bjarne Stroustrup]] para crear C++
- El kernel de [[linux]] de [[linus-torvalds]] está escrito en C
- Base de la [[linea-tiempo-lenguajes-programacion]] y [[fundamentos-programacion]]
- Relacionado con [[grace-hopper]] en la evolución del software
