# Bjarne Stroustrup

Ingeniero de software danés (1950–). Creador del lenguaje de programación C++ (1979) en Bell Labs, extendiendo el lenguaje [[dennis-ritchie|C]] con orientación a objetos, plantillas y programación genérica.

## Aportaciones clave

- **C++ (1979)**: extensión de C con soporte para POO, plantillas y programación genérica
- Filosofía "Zero-overhead abstraction": potencia sin sacrificar rendimiento
- Autor de *The C++ Programming Language* y *A Tour of C++

## Relaciones

- Extendió el lenguaje [[dennis-ritchie|C]] para crear C++
- C++ es mencionado en [[fundamentos-programacion]] y [[linea-tiempo-lenguajes-programacion]]
- Trabajó en Bell Labs junto a [[dennis-ritchie]]
