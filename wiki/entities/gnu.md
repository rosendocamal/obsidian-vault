# GNU

Proyecto iniciado por Richard Stallman en 1983 para crear un sistema operativo completamente libre. El nombre es un acrónimo recursivo de "GNU's Not Unix". Proporcionó las herramientas base (GCC, GNU Bash, GNU Core Utilities) que hacen funcional cualquier distribución Linux.

## Componentes clave

- **GCC**: compilador para C, C++, Fortran y otros
- **GNU Bash**: shell por defecto en la mayoría de distribuciones
- **Core Utilities**: comandos básicos (ls, cp, mv, etc.)
- **GPL**: licencia que garantiza la libertad del software

## Relaciones

- Las herramientas GNU son esenciales en [[fedora]] y cualquier distribución [[linux]]
- Licencia GPL usada por [[git]] y miles de proyectos de código abierto
- Vinculado a la Free Software Foundation y al movimiento de software libre
