# Linus Torvalds

Ingeniero de software finlandés (1969–). Creador del kernel de Linux (1991) y del sistema de control de versiones Git (2005). Su trabajo ha sido fundamental para el movimiento del software de código abierto.

## Aportaciones clave

- **Kernel de Linux (1991)**: núcleo de sistema operativo de código abierto que domina servidores, supercomputadoras, Android e IoT
- **Git (2005)**: sistema de control de versiones distribuido más utilizado del mundo
- Filosofía de desarrollo colaborativo y meritocrático en código abierto

## Relaciones

- Su sistema operativo [[linux]] es la base de servidores, Android y sistemas embebidos
- Relacionado con [[dennis-ritchie]] — el kernel de Linux está escrito en C
- Alternativa de código abierto a los sistemas de [[microsoft]] y [[apple]]
