# Red Hat

Empresa estadounidense de software de código abierto, fundada en 1993. Creadora de Red Hat Enterprise Linux (RHEL) y patrocinadora principal de [[fedora]]. Adquirida por IBM en 2019 por $34 mil millones. (instalacion_fedora_43, 2026)

## Productos principales

- **Red Hat Enterprise Linux (RHEL)**: distribución Linux para empresas
- **Fedora**: distribución comunitaria upstream de RHEL
- **OpenShift**: plataforma Kubernetes para contenedores
- **Ansible**: herramienta de automatización de configuración

## Relaciones

- Patrocinadora de [[fedora]], base de RHEL
- Empresa matriz de Fedora hasta su adquisición por IBM
- Vinculada al ecosistema [[gnu]] y código abierto
