# GitHub

Plataforma de desarrollo colaborativo basada en [[git]]. Adquirida por [[microsoft]] en 2018. Permite alojar repositorios, gestionar issues, pull requests y CI/CD. Es la plataforma de código abierto más grande del mundo.

## Funcionalidades clave

- Repositorios remotos para código fuente
- Pull requests para revisión de código
- GitHub Actions para automatización (CI/CD)
- GitHub Pages para hosting estático gratuito
- Copilot para asistencia de código con IA

## Relaciones

- Funciona con [[git]], creado por [[linus-torvalds]]
- Plataforma remota para [[guia-git-configuracion]] y [[guia-github-nuevo-repo]]
- Adquirida por [[microsoft]]
