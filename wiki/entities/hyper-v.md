# Hyper-V

Hipervisor de virtualización de [[microsoft]]. Disponible en Windows Server y Windows 10/11 Pro. Permite ejecutar máquinas virtuales en el sistema operativo Windows. (virtualbox_fedora_43, 2026)

## Características

- **Hipervisor tipo 1**: se ejecuta directamente sobre el hardware
- Integración nativa con Windows
- Soporte para Linux como sistema operativo invitado
- Gestión mediante PowerShell y Hyper-V Manager

## Relaciones

- Competidor de [[virtualbox]] y [[vmware]]
- Creado por [[microsoft]]
- Alternativa para usuarios de Windows que necesitan virtualización
