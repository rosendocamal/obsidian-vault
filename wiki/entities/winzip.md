# WinZip

Programa de compresión de archivos propietario para Windows, lanzado en 1991. Fue uno de los primeros gestores de archivos ZIP para Windows y popularizó el formato .zip en el escritorio.

## Características

- Soporte para formatos ZIP, 7z, TAR, GZIP
- Integración con Windows Explorer
- Cifrado AES-256
- Sharing en la nube (OneDrive, Google Drive, Dropbox)

## Relaciones

- Competidor de [[seven-zip]] y [[winrar]]
- [[seven-zip]] ofrece funcionalidad similar de forma gratuita
- El formato ZIP es el estándar más universal para compresión
