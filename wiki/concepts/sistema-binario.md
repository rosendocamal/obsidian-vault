# Sistema Binario

Sistema numérico que utiliza solo dos símbolos (0 y 1) para representar cualquier información. Es la base fundamental de la computación digital, pues los circuitos electrónicos pueden representar estos dos estados como apagado/encendido. (algoritmos-booth-multiplicacion-division, 2026)

## Historia

- **Siglo III a.C.**: Pingala usó dos símbolos para métrica poética en la India
- **Siglo XVII**: [[gottfried-leibniz]] formalizó el sistema binario para cálculos mecánicos
- **Siglo XIX**: [[george-boole]] desarrolló el álgebra que relaciona lo binario con la lógica
- **1937**: [[claude-shannon]] aplicó la lógica binaria a circuitos eléctricos

## Sistemas numéricos relacionados

- **Octal**: representación compacta del binario (base 8)
- **Hexadecimal**: codificación eficiente de datos, colores, direcciones de memoria (base 16)
- **Decimal**: puente entre computadora y usuario

## Aplicaciones

- Hardware digital y compuertas lógicas
- El [[algoritmo-booth]] de [[andrew-booth]] para multiplicación binaria
- Codificación de datos, criptografía, redes
- Representación de colores en entornos gráficos
