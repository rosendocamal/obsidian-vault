# Fundamentos de Programación

Base conceptual de la programación de computadoras: algoritmos, variables, estructuras de control, tipos de datos y paradigmas de programación.

## Conceptos clave

- **Algoritmo**: conjunto de pasos ordenados y finitos para resolver un problema
- **Programa**: algoritmo traducido a un lenguaje de programación ejecutable
- **Lenguaje de programación**: reglas y símbolos para escribir programas
- **Compilador**: traduce todo el código a lenguaje máquina antes de ejecutar (C)
- **Intérprete**: traduce y ejecuta línea por línea (Python)

## Paradigmas de programación

- **Imperativo**: instrucciones paso a paso
- **Orientado a objetos**: código organizado en "objetos" del mundo real
- **Funcional**: funciones matemáticas sin mutación de datos

## Estructuras de control

- **Condicionales**: if, if-else, switch
- **Ciclos**: while, for, do-while
- **Tipos de datos**: enteros, flotantes, cadenas, booleanos, arreglos
