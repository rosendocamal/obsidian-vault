# Línea del Tiempo de los Lenguajes de Programación

Evolución cronológica de los lenguajes de programación desde la década de 1940 hasta la actualidad.

## Hitos principales

| Año | Lenguaje | Creador(es) |
|-----|----------|-------------|
| ~1940s | **Ensamblador** | Ingenieros ENIAC |
| 1949 | **Short Code** | John Mauchly |
| 1957 | **Fortran** | John Backus (IBM) |
| 1958 | **Lisp** | John McCarthy (MIT) |
| 1959 | **COBOL** | [[grace-hopper]] |
| 1967 | **Simula** | Dahl y Nygaard |
| 1970 | **Pascal** | Niklaus Wirth |
| 1972 | **C** | [[dennis-ritchie]] |
| 1983 | **C++** | Bjarne Stroustrup |
| 1991 | **Python** | Guido van Rossum |
| 1995 | **Java** | James Gosling (Sun) |
| 1995 | **JavaScript** | Brendan Eich (Netscape) |
| 1995 | **PHP** | Rasmus Lerdorf |
| 2009 | **Go** | Griesemer, Pike, Thompson (Google) |
| 2010 | **Rust** | Graydon Hoare |
| 2011 | **Kotlin** | JetBrains |
| 2014 | **Swift** | Chris Lattner (Apple) |

## Relaciones

- Conecta con [[fundamentos-programacion]] (paradigmas y tipos de lenguajes)
- C y C++ son base del kernel [[linux]] de [[linus-torvalds]]
- Python se usa en [[programacion-orientada-objetos]] y [[algebra-lineal-programacion]]
- La evolución de lenguajes refleja el avance del [[hardware-computacion]]
