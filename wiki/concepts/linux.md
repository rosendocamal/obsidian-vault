# Linux

Sistema operativo de código abierto creado por [[linus-torvalds]] en 1991. Domina en servidores web (>90%), es la base de Android, sistemas IoT y supercomputadoras.

## Estructura en capas

1. **Kernel (núcleo)**: gestiona hardware (CPU, memoria, dispositivos)
2. **Herramientas y aplicaciones**: terminal, navegadores, editores
3. **Interfaz gráfica**: facilita el uso, aunque muchos prefieren la línea de comandos

## Distribuciones (distros)

- **Ubuntu**: fácil de usar, ideal para principiantes
- **Fedora**: innovación y software reciente
- **Debian**: estable, base de Ubuntu
- **Arch**: máxima personalización, para avanzados
- **Red Hat Enterprise Linux**: estabilidad para servidores

## Ventajas

- **Gratuito**: sin licencias
- **Código abierto**: modificable por el usuario
- **Seguridad**: menos vulnerabilidades que otros SO
- **Estabilidad**: ideal para servidores 24/7

## Relaciones

- Creado por [[linus-torvalds]], kernel en C de [[dennis-ritchie]]
- Alternativa a sistemas operativos propietarios
- Base de herramientas en [[fundamentos-programacion]] y [[programacion-orientada-objetos]]
- Usado en servidores para [[investigacion-operaciones]]
