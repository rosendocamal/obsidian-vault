# Simbología Matemática

Conjunto de símbolos, letras (latinas y griegas) y notaciones utilizados para expresar conceptos matemáticos con precisión. Es el lenguaje formal de las matemáticas y la computación. (simbologias-y-planteamientos, 2026)

## Símbolos clave documentados

- **α, β**: variables angulares
- **Σ (Sigma mayúscula)**: sumatoria de una serie
- **σ (Sigma minúscula)**: desviación estándar
- **Δ (Delta mayúscula)**: incremento o diferencia entre variables
- **Ω (Omega mayúscula)**: notación Big-Omega
- **μ (Mu)**: población media
- **x̄**: promedio muestral

## Unidades de almacenamiento (planteamiento adicional)

- Celulares: hasta 2 TB (gama alta 2025)
- HDD: hasta 36 TB (Seagate Exos M)
- Cerebro humano: ~2.5 PB (Scientific American, Paul Reber)
