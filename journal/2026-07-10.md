### Algoritmos

El dominio de la algoritmia requiere comprender la manipulación de datos, la estructura de redes y los paradigmas de resolución de problemas. En el bloque de ordenamiento y acceso, la base se construye transitando desde enfoques clásicos como **Bubble Sort** e **Insertion Sort**, hacia métodos eficientes de división y conquista como **Merge Sort** y **Quick Sort**, la optimización basada en estructuras de **Heap Sort**, y la eficiencia lineal de **Counting Sort**. A la par, la localización de elementos se gestiona mediante **Búsqueda Lineal**, la eficiencia logarítmica de la **Búsqueda Binaria**, y la estructuración dinámica en **Binary Search Trees (BST)**.

Para la modelación de relaciones complejas, los algoritmos de grafos e infraestructura se vuelven indispensables utilizando **BFS** y **DFS** para exploración y conectividad, **Dijkstra** y **Bellman-Ford** para la determinación de caminos mínimos, y **Prim** junto a **Kruskal** para el cálculo de árboles de expansión mínima. Asimismo, el procesamiento avanzado se apoya en la **Exponentiación Binaria** para potencias eficientes, el **Algoritmo de Karatsuba** para multiplicación acelerada, y las técnicas de **KMP (Knuth-Morris-Pratt)** y **Rabin-Karp** para el emparejamiento óptimo de cadenas de texto.

Finalmente, la resolución de problemas complejos de optimización se apoya en tres grandes metodologías de diseño: la **Programación Dinámica (DP)** para resolver subproblemas superpuestos, los **Algoritmos Voraces (Greedy)** para la toma de decisiones óptimas locales, y el **Backtracking** para la exploración sistemática y descarte de soluciones mediante ensayo y error.

####  Fundamentos de Búsqueda y Estructuras Lineales

- [ ] Comprender y codificar la **Búsqueda Lineal** para entender el peor de los casos ($O(n)$) en arreglos no ordenados.
- [ ] Implementar **Bubble Sort** para asimilar mecánicas básicas de intercambio de elementos.
- [ ] Implementar **Insertion Sort** para comprender cómo mantener una sección del arreglo ya ordenada de forma iterativa.
- [ ] Dominar la **Búsqueda Binaria** sobre arreglos previamente ordenados para reducir el tiempo de acceso a $O(\log n)$.

#### Algoritmos Eficientes de División, Conquista y Estructuras Avanzadas

- [ ] Implementar **Merge Sort** para dominar la técnica de dividir, conquistar y fusionar de manera estable.
- [ ] Codificar **Quick Sort** para entender la selección de pivotes y el ordenamiento _in-place_.
- [ ] Estudiar **Counting Sort** como introducción al ordenamiento no comparativo en tiempo lineal.
- [ ] Construir y manipular **Binary Search Trees (BST)** para entender la inserción, eliminación y búsqueda dinámica en árboles.
- [ ] Implementar **Heap Sort** aprendiendo primero a construir y mantener un montículo binario (_max-heap_ o _min-heap_).

#### Exploración de Grafos y Árboles de Expansión

- [ ] Aprender **BFS (Breadth-First Search)** para recorrer grafos por niveles y hallar caminos más cortos en grafos sin pesos.
- [ ] Aprender **DFS (Depth-First Search)** para exploración profunda, detección de ciclos y ordenamiento topológico.
- [ ] Implementar el algoritmo de **Prim** para construir árboles de expansión mínima enfocándose en vértices adyacentes.
- [ ] Implementar el algoritmo de **Kruskal** utilizando la estructura de datos _Disjoint-Set_ (Union-Find) para unir aristas de menor peso.

#### Optimización de Caminos en Grafos

- [ ] Dominar el algoritmo de **Dijkstra** para encontrar caminos mínimos desde un único origen en grafos con pesos no negativos.
- [ ] Estudiar **Bellman-Ford** para resolver caminos mínimos permitiendo pesos negativos y detectando ciclos negativos.

#### Algoritmos Numéricos y Procesamiento de Cadenas

- [ ] Codificar la **Exponentiación Binaria** para calcular potencias en tiempo logarítmico.
- [ ] Comprender el **Algoritmo de Karatsuba** para descubrir cómo reducir la complejidad de la multiplicación de enteros grandes.
- [ ] Estudiar **KMP (Knuth-Morris-Pratt)** para buscar patrones en texto evitando retrocesos innecesarios mediante funciones de prefijo.
- [ ] Implementar **Rabin-Karp** utilizando funciones de _hashing_ para la búsqueda eficiente de múltiples patrones.

#### Paradigmas de Resolución de Problemas y Optimización Compleja

- [ ] Resolver problemas con **Algoritmos Voraces (Greedy)** aprendiendo a tomar la mejor opción local en cada paso.
- [ ] Diseñar soluciones con **Backtracking** para explorar árboles de decisión de forma sistemática y descartar ramas inválidas (ensayo y error).
- [ ] Dominar la **Programación Dinámica (DP)** mediante enfoques _Top-Down_ (memorización) y _Bottom-Up_ (tabulación) para optimizar subproblemas superpuestos.
