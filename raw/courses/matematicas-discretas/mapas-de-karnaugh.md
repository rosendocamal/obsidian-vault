# Mapas de Karnaugh

Rosendo Camal

Los mapas de Karnaugh son un procedimiento gráfico que se utiliza para simplificar funciones booleanas y reducir la complejidad de expresiones lógicas. Su construcción se basa en organizar las combinaciones de variables dentro de una cuadrícula siguiendo el código Gray, lo que permite identificar agrupaciones de valores adyacentes. Con este método se obtienen expresiones más compactas y eficientes, facilitando el diseño y la optimización de sistemas lógicos.

## Resumen

Los mapas de Karnaugh son un método gráfico para simplificar funciones booleanas y reducir su complejidad. Se construyen en una cuadrícula con código Gray, donde se agrupan los unos en potencias de dos para obtener términos simplificados. Este procedimiento genera expresiones más compactas, disminuye operaciones y optimiza la representación ordenada de sistemas lógicos, conectando la teoría del álgebra booleana con la eficiencia en el diseño de circuitos lógicos.

## Ideas principales

Los mapas de Karnaugh son un método gráfico para simplificar funciones booleanas. Se construyen en una cuadrícula con código Gray. Cada celda se llena con 1, 0 o X. La simplificación se logra agrupando unos en potencias de dos. El resultado son expresiones más compactas y eficientes; y se aplican en el diseño de circuitos.

## Conceptos importantes y difíciles

- **Mapa de Karnaugh**: representación gráfica para simplificar funciones booleanas.
- **Función booleana**: expresión lógica compuesta por variables binarias y operaciones.
- **Código Gray**: sistema de codificación donde solo cambia un bit entre valores consecutivos.
- **Minitérmino**: cada celda del mapa que representa una combinación específica de variables.
- **Agrupación en potencias de dos**: regla fundamental para simplificar (1, 2, 4, 8, etc.).
- **Condiciones de indiferencia (X)**: valores que pueden ser 0 o 1 sin afectar la función.
- **Eliminación de variables**: proceso donde las variables que cambian dentro de una agrupación se eliminan.
- **Función lógica simplificada**: resultado final con menos términos y variables.
