# C++

Lenguaje de programación de propósito general creado por Bjarne Stroustrup en 1979 en Bell Labs como extensión del lenguaje C, añadiendo programación orientada a objetos.

## Características

- Programación orientada a objetos
- Tipado estático, compilado
- Multiplataforma
- Permite programación de bajo y alto nivel
- Soporta programación genérica con templates
- Gestión manual de memoria

## Ventajas

- Alto rendimiento y eficiencia
- Control total sobre recursos del sistema
- Amplia biblioteca estándar (STL)
- Portabilidad entre plataformas
- Gran comunidad y recursos disponibles
- Versatilidad para diferentes tipos de proyectos

## Desventajas

- Curva de aprendizaje pronunciada
- Sintaxis compleja

## Aplicaciones

- Sistemas operativos (Windows, Linux)
- Navegadores web (Chrome, Firefox)
- Motores de videojuegos (Unreal Engine, Unity)
- Software de bases de datos
- Finanzas
