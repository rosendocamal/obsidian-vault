## 1. Lenguaje de Programación

Un lenguaje de programación es un conjunto de reglas que permite a los desarrolladores escribir instrucciones para que una computadora ejecute tareas. Los lenguajes de programación varían en sintaxis y paradigmas, como imperativo, orientado a objetos y funcional.

**Ejemplos comunes:**

- **C:** Eficiente y de bajo nivel.
- **Java:** Usado en aplicaciones empresariales y móviles.
- **Python:** Sencillo, usado en ciencia de datos y web.
- **JavaScript:** Para sitios web interactivos.

**Tipos de lenguajes:**

- **Compilados:** Traducción completa antes de ejecutar (ej. C).
- **Interpretados:** Ejecución línea por línea (ej. Python).

Los lenguajes de programación permiten crear software, aplicaciones y resolver problemas específicos.

## 2. ¿Qué es un IDE?

Un IDE (Integrated Development Environment), o Entorno de Desarrollo Integrado, es una aplicación que proporciona herramientas esenciales para el desarrollo de software en un solo lugar. Su objetivo principal es facilitar la creación, edición, prueba y depuración de programas de software.

**Características comunes de un IDE:**

- **Editor de código:** Permite escribir y modificar el código fuente.
- **Compilador o intérprete:** Traduce el código fuente a un lenguaje que la computadora pueda entender.
- **Depurador (debugger):** Ayuda a detectar errores en el código.
- **Sistema de gestión de proyectos:** Permite organizar archivos y directorios de proyectos.
- **Herramientas de integración:** Algunas incluyen integración con bases de datos, servidores o servicios en la nube.

**Ejemplos de IDEs populares:**

- Visual Studio Code
- Eclipse
- IntelliJ IDEA
- PyCharm
- Xcode

## 3. ¿Qué son las tecnologías de la información y de la comunicación (TIC)?

Las TIC (Tecnologías de la Información y de la Comunicación) hacen referencia al conjunto de herramientas, sistemas y recursos tecnológicos utilizados para gestionar, procesar, almacenar y transmitir información. Las TIC engloban tanto el uso de computadoras como de redes de comunicación, tales como internet, telefonía, radio, televisión, y otros medios de comunicación digital.

**Ámbitos de las TIC:**

- **Tecnología de la información (TI):** Se refiere al uso de hardware y software para procesar, almacenar y transmitir datos.
- **Tecnología de la comunicación (TC):** Se centra en el uso de medios y dispositivos que permiten la comunicación a distancia, como el correo electrónico, las redes sociales, la telefonía móvil y la televisión.

**Importancia de las TIC:**

- **Educación:** Herramientas digitales para aprender y enseñar.
- **Trabajo:** Facilitación del trabajo remoto, gestión empresarial y productividad.
- **Salud:** Mejora de la atención médica mediante la telemedicina y la gestión electrónica de datos médicos.
- **Entretenimiento:** Videojuegos, películas y otros contenidos digitales.

## 4. ¿Qué es un lenguaje de marcas (HTML)?

Un lenguaje de marcas es un tipo de lenguaje de programación que se utiliza para estructurar y presentar contenido en documentos electrónicos. Estos lenguajes no están diseñados para realizar cálculos o procesos lógicos como los lenguajes de programación tradicionales, sino para definir el formato y la estructura de los contenidos.

HTML (HyperText Markup Language) es el lenguaje de marcas más conocido y utilizado para la creación de páginas web. HTML utiliza una serie de etiquetas o "tags" que indican cómo debe estructurarse el contenido, como textos, imágenes, enlaces, tablas, entre otros.

**Características del HTML:**

- **Etiquetas:** HTML usa etiquetas delimitadas por corchetes angulares, como `<p>` para párrafos o `<a>` para enlaces.
- **Estructura jerárquica:** El contenido de una página web se organiza de manera jerárquica utilizando contenedores y etiquetas dentro de otros contenedores.
- **Semántica:** Existen etiquetas específicas que indican el significado del contenido, como `<h1>` para encabezados, `<footer>` para pies de página, etc.

**Ejemplo básico de HTML:**

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Primera Página</title>
</head>
<body>
    <h1>Bienvenidos a mi página web</h1>
    <p>Este es un párrafo de texto en HTML.</p>
    <a href="https://www.ejemplo.com">Enlace a otro sitio</a>
</body>
</html>
```

Este ejemplo crea una página web con un encabezado, un párrafo y un enlace.
