# Fundamentos de Programación: Conceptos Básicos

## Clasificación de software

- **Software de Sistema:** Conjunto de programas que gestionan los recursos del hardware. Ejemplos: sistemas operativos como Linux, Windows, Android, iOS y macOS.
- **Software de Aplicación:** Programas diseñados para realizar tareas específicas del usuario. Ejemplo: navegadores, editores de texto, videojuegos.

## Términos fundamentales

### Algoritmo

Conjunto de pasos ordenados y finitos para resolver un problema o realizar una tarea.

### Editor de Texto

Programas que permiten escribir y editar código.

### Programa

Es un algoritmo traducido a un lenguaje de programación que puede ejecutarse por una computadora.

### Lenguaje de Programación

Es un conjunto de reglas y símbolos que permiten escribir programas que una computadora puede entender y ejecutar.

### Paradigmas de Programación

Son diferentes formas o estilos de pensar y escribir el código en un programa. Cada uno tiene su propia manera de resolver problemas.

- **Imperativo:** El programa se escribe paso por paso, como seguir instrucciones en una receta.
- **Orientado a Objetos:** Organiza el código en "objetos" que representan cosas del mundo real, como un auto o un usuario.
- **Funcional:** Usa funciones matemáticas y evita cambiar datos una vez que se crean.

### Programación

Proceso de diseñar, codificar y probar programas que resuelvan problemas o realicen tareas, es decir, crear algoritmos usando lenguajes de programación.

## Compiladores e Intérpretes

- **Compilador:** Traduce todo el código a lenguaje máquina (binario) antes de ejecutarlo, como C.
- **Intérprete:** Traduce y ejecuta línea por línea, como Python.

Ambos permiten ejecutar programas pero de distinta manera.

## Ejecutables

Archivos resultantes de la compilación de un programa que se pueden ejecutar directamente en el sistema operativo, como los archivos `.exe` en Windows.

## Consola de Línea de Comandos

Programa que mediante texto permite ejecutar instrucciones en el sistema operativo. Ejemplos sería CMD en Windows y la Terminal en Linux.

## Variables y Constantes

- **Variable:** Es la asignación en memoria de un dato que puede cambiar durante el uso de un programa.
- **Constante:** Es la asignación en memoria de un valor que no cambia durante el uso de un programa.
