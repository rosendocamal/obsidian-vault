# C++ — Estructuras de Control y Datos

## Condicionales

Permiten que el programa tome decisiones y ejecute diferentes bloques de código según si una condición se cumple o no.

- **if:** Ejecuta un bloque de código si una condición es verdadera.
- **if-else:** Permite ejecutar un bloque de código si la condición es verdadera (if) y otro si es falsa (else).
- **switch:** Se usa para seleccionar uno de muchos bloques de código a ejecutar, ideal para evaluar una única variable contra múltiples valores constantes.

## Bucles

Permiten repetir la ejecución de un bloque de código varias veces.

- **while:** Repite un bloque de código mientras una condición sea verdadera. La condición se evalúa al principio.
- **for:** Se utiliza cuando se conoce de antemano el número exacto de repeticiones.
- **do-while:** Similar al ciclo while, pero garantiza que el bloque de código se ejecute al menos una vez, ya que la condición se evalúa al final.

## Estructuras de Datos

Son colecciones que permiten organizar y gestionar datos de manera eficiente.

- **Arrays y Vectores:** Arrays son colección de elementos del mismo tipo y tamaño fijo. Vectores son colección de elementos del mismo tipo, pero con tamaño dinámico, más flexibles que los arrays.
- **Matrices:** Un array de dos o más dimensiones, útil para representar tablas o cuadrículas de datos.
- **Punteros:** Una variable que almacena la dirección de memoria de otra variable, permitiendo la manipulación directa de la memoria.
