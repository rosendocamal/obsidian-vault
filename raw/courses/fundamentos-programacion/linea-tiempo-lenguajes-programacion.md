# Línea del Tiempo de los Lenguajes de Programación

| Año | Lenguaje | Descripción |
|---|---|---|
| ~1940s | **Ensamblador** | Desarrollado por diversos ingenieros en las primeras computadoras electrónicas como la ENIAC. |
| 1949 | **Short Code** | Considerado uno de los primeros lenguajes de alto nivel y creado por John Mauchly para UNIVAC. |
| 1957 | **Fortran** | Lenguaje de alto nivel diseñado para cálculos científicos y matemáticos. Creado por John Backus en IBM. |
| 1958 | **Lisp** | Lenguaje funcional y recursivo, usado principalmente para IA e introdujo listas como estructura de datos clave. Creado por John McCarthy en el MIT. |
| 1959 | **COBOL** | Lenguaje orientado a la programación empresarial, especialmente para procesamiento de datos y transacciones. Creado por Grace Hopper. |
| 1967 | **Simula** | Primer lenguaje orientado a objetos, introdujo conceptos clave como clases y objetos. Creado por Ole-Johan Dahl y Kristen Nygaard. |
| 1970 | **Pascal** | Lenguaje de programación estructurada diseñado para enseñar buenas prácticas de programación y desarrollo académico. Creado por Niklaus Wirth. |
| 1972 | **C** | Lenguaje de propósito general, base del SO UNIX. Fue clave para el desarrollo de software de alto rendimiento. Creado por Dennis Ritchie. |
| 1983 | **C++** | Extensión de C que incorpora programación orientada a objetos, muy utilizado en aplicaciones de alto rendimiento y videojuegos. Creado por Bjarne Stroustrup. |
| 1991 | **Python** | Lenguaje fácil de aprender y usar, diseñado para la productividad. Popular en ciencia de datos, IA y desarrollo web. Creado por Guido van Rossum. |
| 1995 | **Java** | Lenguaje multiplataforma y orientado a objetos. Creado por James Gosling en Sun Microsystems. |
| 1995 | **JavaScript** | Lenguaje de scripting para interactuar con el navegador web, fundamental en el desarrollo de aplicaciones web modernas. Creado por Brendan Eich en Netscape. |
| 1995 | **PHP** | Lenguaje de programación para desarrollo web dinámico, ampliamente utilizado en plataformas como WordPress y Facebook en sus inicios. Creado por Rasmus Lerdorf. |
| 2009 | **Go** | Lenguaje eficiente y sencillo para la programación concurrente y la creación de servicios en la nube. Creado por Robert Griesemer, Rob Pike y Ken Thompson en Google. |
| 2010 | **Rust** | Lenguaje enfocado en la seguridad, el rendimiento y la concurrencia. Popular para apps de bajo nivel y sistemas de alto rendimiento. Creado por Graydon Hoare. |
| 2011 | **Kotlin** | Lenguaje para móvil interoperable con Java y adoptado en Android. Creado por JetBrains. |
| 2014 | **Swift** | Lenguaje moderno para el desarrollo de aplicaciones iOS y macOS, diseñado para ser rápido y seguro. Creado por Chris Lattner en Apple. |
